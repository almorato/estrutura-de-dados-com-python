## Perguntas

### 1. Qual é a principal diferença entre tupla e lista?
a) Tupla suporta mais tipos de dados
b) Tupla é imutável — não pode ser alterada após criada
c) Tupla é mais lenta que lista
d) Tupla não mantém a ordem dos itens

### 2. O que acontece ao criar o conjunto `{1, 2, 2, 3, 3, 3}`?
a) Erro — conjuntos não aceitam repetição
b) `{1, 2, 2, 3, 3, 3}` — mantém tudo
c) `{1, 2, 3}` — remove duplicatas automaticamente
d) `{3, 3, 3, 2, 2, 1}` — inverte a ordem

### 3. Dado `a = {1, 2, 3}` e `b = {2, 3, 4}`, qual é o resultado de `a & b`?
a) `{1, 2, 3, 4}`
b) `{1, 4}`
c) `{2, 3}`
d) `{1, 2, 3, 2, 3, 4}`

### 4. Quando faz mais sentido usar uma tupla em vez de uma lista?
a) Quando precisar adicionar itens frequentemente
b) Quando os dados representam um registro fixo, como coordenadas
c) Quando precisar ordenar os itens
d) Quando os itens forem de tipos diferentes

### 5. Qual código remove duplicatas de uma lista da forma mais simples?
a) `lista.remove_duplicates()`
b) `list(set(lista))`
c) `lista.unique()`
d) `dict(lista)`
