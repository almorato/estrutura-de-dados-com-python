## Perguntas

### 1. O que melhor define uma estrutura de dados?
a) Um tipo de banco de dados relacional
b) Uma forma organizada de guardar e acessar informações no computador
c) Uma linguagem de programação específica
d) Um algoritmo de ordenação

### 2. Qual analogia melhor representa uma pilha (stack)?
a) Uma fila do banco
b) Uma agenda telefônica
c) Uma pilha de pratos
d) Um caderno de anotações

### 3. O que significa dizer que uma estrutura é "mutável"?
a) Ela pode ser copiada para outra variável
b) Ela pode ser alterada depois de criada
c) Ela ocupa menos memória
d) Ela é mais rápida que outras estruturas

### 4. Qual estrutura seria mais adequada para guardar o nome e o telefone de contatos?
a) Pilha
b) Conjunto
c) Dicionário
d) Tupla

### 5. Por que a escolha da estrutura de dados certa importa?
a) Porque algumas estruturas só funcionam em Python 3
b) Porque afeta a velocidade, uso de memória e clareza do programa
c) Porque o Python exige o uso de estruturas específicas
d) Não importa, qualquer estrutura funciona igual
