## Perguntas

### 1. O que significa LIFO?
a) Last Input, First Output
b) Last In, First Out
c) List In, First Out
d) Last In, Fixed Order

### 2. Em uma pilha implementada com lista Python, qual método é usado para empilhar (push)?
a) `insert()`
b) `push()`
c) `append()`
d) `add()`

### 3. Você empilhou: "A", "B", "C" nessa ordem. Após dois `pop()`, o que sobrou na pilha?
a) "C"
b) "B"
c) "A"
d) A pilha ficou vazia

### 4. Qual operação do dia a dia é melhor representada por uma pilha?
a) Fila de impressão de documentos
b) Lista de contatos em ordem alfabética
c) Ctrl+Z para desfazer ações
d) Agenda com horários marcados

### 5. Como verificar o topo de uma pilha sem remover o item?
a) `pilha.pop()`
b) `pilha[0]`
c) `pilha[-1]`
d) `pilha.peek()`
