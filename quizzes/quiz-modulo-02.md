## Perguntas

### 1. Qual método adiciona um item ao final de uma lista?
a) `insert()`
b) `add()`
c) `append()`
d) `push()`

### 2. O que faz `lista.pop(0)`?
a) Remove o último item da lista
b) Remove o item no índice 0 (primeiro item)
c) Retorna o primeiro item sem remover
d) Esvazia a lista

### 3. Dado `frutas = ["maçã", "banana", "laranja"]`, qual é o valor de `frutas[-1]`?
a) "maçã"
b) "banana"
c) "laranja"
d) Erro — índice inválido

### 4. Qual das opções abaixo NÃO é uma característica de listas em Python?
a) São mutáveis
b) Podem conter itens de tipos diferentes
c) São imutáveis após criadas
d) Mantêm a ordem de inserção

### 5. Você tem uma lista com 1000 nomes e precisa verificar frequentemente se um nome específico está nela. Qual estrutura seria mais eficiente para isso?
a) Manter como lista
b) Converter para conjunto (set)
c) Converter para tupla
d) Usar uma pilha
