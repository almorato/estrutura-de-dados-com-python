## Perguntas

### 1. Você precisa armazenar o histórico de páginas visitadas no browser para implementar o botão "voltar". Qual estrutura usar?
a) Fila
b) Pilha
c) Conjunto
d) Árvore

### 2. Um sistema de suporte técnico precisa atender chamados na ordem em que chegaram. Qual estrutura usar?
a) Pilha
b) Dicionário
c) Fila
d) Tupla

### 3. Você tem uma lista com 10.000 IDs de usuários e precisa verificar rapidamente se um ID específico existe. Qual estrutura é mais eficiente?
a) Lista
b) Tupla
c) Conjunto
d) Pilha

### 4. Um sistema precisa guardar as notas de cada aluno pelo número de matrícula. Qual estrutura é mais adequada?
a) Lista de tuplas
b) Dicionário
c) Conjunto
d) Árvore binária

### 5. Quais são as coordenadas fixas (latitude, longitude) de uma cidade? Qual estrutura armazena isso de forma mais semântica?
a) Lista
b) Dicionário
c) Tupla
d) Conjunto
