# Estrutura de Dados com Python

Curso prático para quem tem conhecimento básico de Python e quer aprender a implementar estruturas de dados fundamentais.

## Como iniciar

1. Clone este repositório
2. Abra no Claude Code dentro desta pasta
3. Cole a mensagem abaixo:

> Olá! Quero iniciar o curso de Estrutura de Dados com Python.

## O que você vai aprender (8 módulos)

1. Introdução às Estruturas de Dados
2. Listas e Operações Fundamentais
3. Pilhas (Stacks)
4. Filas (Queues)
5. Tuplas e Conjuntos (Sets)
6. Dicionários e Tabelas Hash
7. Árvores Binárias
8. Projeto Final — Escolhendo a Estrutura Certa

## O que tem neste repositório

- `modulos/` — conteúdo de cada módulo
- `quizzes/` — quiz com 5 perguntas por módulo
- `progresso.md` — rastreie seu avanço no curso
- `docs/` — site GitHub Pages para navegar pelo curso
- `CLAUDE.md` — instruções do instrutor

## Site do curso

Após publicar no GitHub Pages, o curso fica disponível como minisite navegável.

Para publicar: **Settings → Pages → Source: main, pasta /docs**
