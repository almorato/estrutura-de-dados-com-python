# Glossário

**Árvore Binária** — Estrutura hierárquica onde cada nó tem no máximo dois filhos.

**BST (Binary Search Tree)** — Árvore Binária de Busca: valores menores à esquerda, maiores à direita.

**Complexidade de tempo** — Medida de como o tempo de execução cresce conforme o volume de dados aumenta.

**deque** — Double-ended queue. Estrutura do Python otimizada para inserções e remoções em ambas as extremidades.

**Dicionário** — Estrutura de dados que armazena pares chave → valor.

**FIFO** — First In, First Out. O primeiro que entra é o primeiro que sai (filas).

**Folha (leaf)** — Nó de uma árvore que não tem filhos.

**Imutável** — Objeto que não pode ser alterado após criado (ex: tupla).

**LIFO** — Last In, First Out. O último que entra é o primeiro que sai (pilhas).

**Lista** — Coleção ordenada e mutável de itens em Python.

**Mutável** — Objeto que pode ser alterado após criado (ex: lista, dicionário).

**Nó (node)** — Cada elemento de uma estrutura como árvore ou lista encadeada.

**Pilha (Stack)** — Estrutura LIFO. O último item adicionado é o primeiro a ser removido.

**Raiz (root)** — O nó do topo de uma árvore, sem pai.

**Set (Conjunto)** — Coleção sem ordem e sem itens repetidos.

**Tabela Hash** — Mecanismo interno do dicionário Python que permite acesso por chave em tempo constante.

**Tupla** — Coleção ordenada e imutável de itens em Python.
