# Quizzes

Cada módulo tem um quiz com 5 perguntas de múltipla escolha.

## Como fazer o quiz interativo

Abra o Claude Code dentro da pasta do curso e diga:
> "Quero fazer o quiz do módulo X"

O Claude vai apresentar uma pergunta por vez, corrigir suas respostas e atualizar seu progresso automaticamente.
