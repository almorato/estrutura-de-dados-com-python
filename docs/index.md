# Estrutura de Dados com Python

Curso prático para quem tem conhecimento básico de Python e quer aprender a implementar as principais estruturas de dados.

## Objetivo

Ao terminar este curso, você será capaz de implementar estruturas de dados básicas em Python e escolher a estrutura certa para cada situação.

## Público-alvo

Pessoas com conhecimento básico de programação em Python.

## Como estudar

Este curso é conduzido pelo Claude no terminal. Para estudar:

1. Clone o repositório
2. Abra no Claude Code dentro da pasta do curso
3. Cole a mensagem: `Olá! Quero iniciar o curso de Estrutura de Dados com Python`

O Claude assume como instrutor e conduz você pelos módulos.
