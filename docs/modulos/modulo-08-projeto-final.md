# Módulo 8 — Projeto Final: Escolhendo a Estrutura Certa

## Guia de decisão

| Estrutura | Use quando |
|---|---|
| Lista | Sequências ordenadas que mudam |
| Pilha | Desfazer ações, histórico LIFO |
| Fila | Processar em ordem de chegada |
| Tupla | Dados fixos e imutáveis |
| Conjunto | Eliminar duplicatas, comparar grupos |
| Dicionário | Buscar por identificador (chave) |
| Árvore | Busca rápida em volume, hierarquias |
