# Módulo 7 — Árvores Binárias

## O que é?

Estrutura hierárquica onde cada nó tem no máximo dois filhos.

```
        10
       /  \
      5    15
     / \     \
    3   7     20
```

## Regra da Árvore Binária de Busca

- Valores menores → esquerda
- Valores maiores → direita

Isso permite encontrar qualquer valor sem checar tudo.

## Percurso em ordem

Percorrer em ordem (esquerda → raiz → direita) retorna os valores em ordem crescente.

## Quando usar

- Busca rápida em grande volume de dados
- Encontrar valores em um intervalo
- Representar hierarquias (categorias, menus)
