# Módulo 1 — Introdução às Estruturas de Dados

## O que é uma estrutura de dados?

Estrutura de dados é uma forma organizada de guardar e acessar informações no computador.

Pense assim: você pode guardar suas roupas jogadas no chão (funciona, mas é difícil achar algo) ou organizadas em gavetas por tipo. Estruturas de dados são essas "gavetas" para o seu programa.

## Por que isso importa?

Com a estrutura certa, seu programa encontra informações mais rápido, usa menos memória e fica mais fácil de entender.

## As estruturas do curso

| Estrutura | Analogia |
|---|---|
| Lista | Caderno de anotações |
| Pilha | Pilha de pratos |
| Fila | Fila do banco |
| Tupla | Endereço de uma casa |
| Conjunto | Lista de convidados únicos |
| Dicionário | Agenda telefônica |
| Árvore | Organograma de empresa |

## Conceitos importantes

**Mutável** — pode ser alterado depois de criado (ex: lista).  
**Imutável** — não pode ser alterado depois de criado (ex: tupla).
